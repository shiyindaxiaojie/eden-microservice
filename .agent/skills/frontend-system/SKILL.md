# Frontend System Module Skill

This skill allows you to work on the System module in the frontend.

## 1. Scope
- **User Management**: User list, add/edit user, assign roles.
- **Role Management**: Role list, permission assignment.
- **Menu Management**: Menu tree editor.
- **Scheduled Tasks**: Task list and logs.

## 2. Key Paths
- **Views**: `web/src/views/system/`
- **API**: `web/src/api/system/`
- **Routes**: Define in `web/src/router/modules/system.ts`.

## 3. Dependencies
- Depends on `frontend-common` (Element Plus, Axios).
