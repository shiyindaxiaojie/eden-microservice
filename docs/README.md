# Documentation

This directory contains the operator-facing documentation set for Focalors.

## Document Structure

1. [Deployment](./deployment.md)
   Covers standalone nodes, AP clusters, CP clusters, and the corresponding configuration differences.
2. [Integration](./integration.md)
   Describes the supported integration paths for Go services, HTTP clients, gRPC clients, and migration from Nacos or Consul.
3. [Architecture](./architecture.md)
   Defines system boundaries, runtime layers, protocol responsibilities, and the public API surface.
